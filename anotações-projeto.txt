2º método GET e POST

    Para recuperar os parametros que são exibidos na URL utilizamos a super global "$_GET", como a super global é um array, passamos os indices
    que são exibidos na URL e passamos dentro da array "$_GET['email'], $_get['senha']".

    A resposta dada pelo servidor apache não é o script PHP, ele processa o script PHP e exibe o resultado da interpretação do script que é
    submetido para o apache que é o responsavel por controlar a resposta enviada para o browser. O browser sempre vai receber a resposta em HTML
    e não em PHP.

    Para escondermos os parametros que são exibidos na URL utilizamos a super global "$_POST" e devemos passar como parametro no <form></form>
    do codigo HTML e depois assim chamar no arquivo onde vai ta contigo o codigo PHP.
